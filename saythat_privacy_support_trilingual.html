隐私声明与使用说明

最近更新：2025年10月

1. 产品简介 (BYOK模式)

SayThat 是一款语音转写与润色工具。应用本身不附带任何云端模型密钥。

  • 所有云请求均使用您自行提供的 API Key (Google Gemini 或 OpenAI)。
  • 请求直接连接官方服务，不经过开发者的服务器。

2. 我们处理的数据

  • 音频内容：发送至 Google Gemini 和 OpenAI 用于处理。
  • 文本内容：来自 Google Gemini 和 OpenAI 生成结果，以及您自行添加的内容。
  • 本地存储：生成的笔记、缓存音频及偏好设置均保存在您的设备上。
  • 不收集：我们不收集广告ID、位置信息、通讯录，且无后台跟踪代码。

3. 数据的用途与去向

您的数据通过 HTTPS 直接发送至您选择的第三方服务（Google Gemini 或 OpenAI）。开发者无法查看、存储或分享您的数据。

所有转写结果与缓存文件存储在设备本地的 “On My iPhone” 目录，您拥有完全控制权，可随时导出或删除。

4. 权限与用户控制

  • 麦克风：仅在您点击录音时使用。
  • 数据清除：可在设置中一键清除语音缓存，或通过删除笔记移除关联音频。
  • API Key 管理：您可以随时更换、删除保存在本地的 API Key。

5. 第三方服务说明

应用与以下服务交互（取决于您填写的 Key），请遵循其隐私政策：

  • Google Gemini API
  • OpenAI API

6. 儿童与敏感数据

本应用面向 13+ 用户，不专门收集儿童数据。应用不处理健康、金融等敏感类别；若您自行录入此类信息，应用仅按上述用途进行本地处理或API转发。

Privacy Policy & Support

Last Updated: October 2025

1. Product Introduction (BYOK)

SayThat is a tool for voice transcription and translation. The app does not include any cloud model keys.

  • All cloud requests use the API Key you provide (Google Gemini or OpenAI).
  • Requests connect directly to official services and do not pass through the developer's server.

2. Data Handling

  • Audio Content: Sent to Google Gemini and OpenAI for processing.
  • Text Content: Derived from Google Gemini and OpenAI outputs, and your own additions.
  • Local Storage: Notes, cached audio, and preferences are stored on your device.
  • No Collection: We do not collect Ad IDs, location, contacts, or perform background tracking.

3. Data Usage & Destination

Your data is sent via HTTPS directly to the third-party service you choose. The developer does not proxy, store, or share your data.

All results are stored locally in the "On My iPhone" directory, giving you full control to export or delete them at any time.

4. User Control

  • Microphone: Accessed only when you initiate recording.
  • Data Clearing: You can clear voice caches in Settings or delete notes to remove associated audio.
  • API Key Management: You can replace or delete your stored API Keys at any time.

5. Third-Party Services

Interaction with these services relies on your keys and is subject to their privacy policies:

  • Google Gemini API
  • OpenAI API
